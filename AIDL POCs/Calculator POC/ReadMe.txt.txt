1. this app is a Client-Server architecture where client sends a request to the AIDL Service and Server receives it from the AIDL and return the Computed output back to AIDL.
2. This app allows user to enter two integer numbers and find their sum, Difference, Multiplication and Division
3. Implemented AIDL Service as a middleware which communicates with client and server app
4. based on the operator which got clicked by the user client communicate with the specific AIDL Service like for + it communicate with getSUM AIDL Service and so on
